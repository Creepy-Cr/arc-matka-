# Overview

Arc Matka is a wallet-based card result game built around scheduled sessions and clear result rules

Users enter a session, choose a bet type and track the outcome through their connected wallet

## Core Ideas

- Wallet-based account access
- Scheduled sessions
- Card-based result calculation
- Clear bet categories
- My Bets tracking
- Claim flow for eligible rewards

## What Users See

Users see available sessions, betting status, card result details and bet history. If a session is not available, the interface shows a clear status such as Awaiting schedule or Session Cancelled

## What This Public Package Covers

This documentation explains the product at a user level. It does not include private source code or sensitive product operations
