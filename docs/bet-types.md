# Bet Types

Arc Matka supports several bet types. Potential payout is shown before wallet confirmation

## Single Number

Choose a digit from 0 to 9. The bet wins if the selected digit matches the final result number

Example: select `5`. If the final result is `5`, the bet wins

## Jodi

Jodi is made from the Open result and Close result

The Open result is the first digit. The Close result is the second digit

Example: Open result `4` and Close result `8` makes `48`

Example: Open result `0` and Close result `7` makes `07`

## Unique 3 Cards

Choose three different digits. The code is checked in ascending order

Examples: `123`, `269`

Invalid examples: `112`, `555`

## Pair

Choose a three digit code where one digit appears twice and one digit is different. Order does not matter because the code is checked in ascending order

Examples: `224`, `445`

## Three of Kind

Choose a code where all three digits are the same

Examples: `111`, `555`
