# Fairness

Arc Matka fairness is explained in user-facing terms through fixed schedules, locked betting windows and published result rules

## Betting Windows

Betting closes before the result reveal. Users cannot place a new bet after the betting window is closed

## Published Result Rules

Card values follow the same public calculation rules every time:

- Ace = 1
- 2 to 9 = same value
- 10, J, Q and K = 10

The final result is the last digit of the total

## Wallet-Linked Bets

Confirmed bets stay linked to the wallet that placed them

## Result Publication

Results are published after reveal. Users can review result status and their own bet status through the product interface

## Careful Randomness Statement

This public documentation does not claim fully verified live randomness unless that status is confirmed in official product materials. Users should rely on the current product status shown in the live interface
