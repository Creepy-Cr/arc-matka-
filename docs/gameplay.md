# Gameplay

Arc Matka follows a simple session flow

## User Flow

1. Connect wallet
2. Sign in with wallet
3. Choose a session
4. Review timing and bet details
5. Place a bet
6. Wait for the result
7. Check My Bets
8. Claim rewards if eligible

## Wallet Sign-In

Signing in confirms wallet ownership. It does not place a bet and it does not move funds

Placing a bet is a separate wallet transaction that the user must review and confirm

## Before Placing a Bet

Users should check:

- Session name
- Session status
- Betting close time
- Bet type
- Selection
- Stake
- Potential payout

Confirmed bets cannot be cancelled
