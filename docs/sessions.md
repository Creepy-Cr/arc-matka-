# Sessions

Arc Matka uses scheduled sessions. Each session has its own timing and status

## Session Types

### Open Session

The first main session of the day

### Close Session

A later session. The Open result and Close result are also used together for Jodi

### Midnight Session

A late session with its own schedule

## Session Status

### Awaiting schedule

No current session is available for that market yet

### Betting open

The session is accepting bets

### Betting closed

The session is no longer accepting new bets

### Awaiting reveal

Betting has ended and the result has not been shown yet

### Session cancelled

The session cannot be entered and no new bet can be placed

## Time Display

Session times are shown in the user browser timezone where supported by the product interface
