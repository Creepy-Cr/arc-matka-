# Card Results

Arc Matka results are based on three revealed cards

## Card Values

- Ace = 1
- 2 to 9 = same value
- 10, J, Q and K = 0

## Final Result Number

The three card values are added. The last digit of the total is the final result number

## Examples

### Example 1

Cards: 2, 5 and 7

Calculation: `2 + 5 + 7 = 14`

Final result: `4`

### Example 2

Cards: A, 9 and K

Calculation: `1 + 9 + 0 = 10`

Final result: `0`

## 3-Card Code

The three digit values are arranged in ascending order for official checking

Example: `321` is checked as `123`

Example: `454` is checked as `445`
