# Roadmap

This roadmap describes public product direction. It does not promise dates

## Current Focus

- Public docs
- Mobile polish
- Profile improvements
- Result history improvements
- Verified randomness integration when official provider details are available
- UX improvements

## Documentation

Public docs will continue to improve with clearer user guides, examples and FAQ updates

## Mobile Experience

Mobile layout and readability will continue to be refined for smaller screens

## Profile Experience

Profile pages may receive clearer account views, better activity summaries and richer user-facing preferences

## Result History

Result history may receive improved filtering, clearer status labels and better wallet-linked activity views

## Verified Randomness

Verified randomness integration depends on official provider details becoming available for the target network. No provider address is assumed in this public roadmap
