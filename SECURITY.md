# Security Policy

## Reporting Security Issues

Please do not create a public issue for sensitive security reports

Send reports to:

security-contact@example.com

Include a clear description, affected area, steps to reproduce if safe and any relevant screenshots or transaction references

## Scope

This public repository contains documentation and product materials only. It does not contain the private application source or privileged product code

## Response

The Arc Matka team will review reports and respond when possible. No reward or bounty is promised unless a separate written program states otherwise

## Safe Reporting

Do not attempt to access wallets, user accounts, funds or private systems. Do not publish sensitive details before the team has reviewed the report
