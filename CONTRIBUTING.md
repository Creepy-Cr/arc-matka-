# Contributing

Thank you for your interest in Arc Matka

This public repository accepts documentation issues, typo fixes, product feedback and user-facing suggestions only

## Accepted Contributions

- Documentation corrections
- FAQ improvements
- Clearer gameplay explanations
- Public product feedback
- Accessibility and readability suggestions

## Not Accepted Here

Implementation contributions are not accepted in this repository because the source is maintained privately

Please do not submit application code, smart contract code, server code, wallet session code or private configuration material

## Pull Request Guidelines

- Keep language simple and user-facing
- Avoid overpromising
- Do not include secrets or environment values
- Do not include private product details
- Use clear commit messages
